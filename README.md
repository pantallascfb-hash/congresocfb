# Nuxcrew GT

**Tu socio ideal, automatizando flujos y sistemas de tu negocio.**

Landing page oficial de Nuxcrew — empresa guatemalteca de ingeniería de software y consultoría tecnológica.

---

## Servicios

- **Software con IA** — Apps web/móvil, automatización con IA, procesamiento de documentos, NLP, chatbots.
- **QA Engineering** — Pruebas automatizadas, E2E, CI/CD, pruebas de carga, suites de regresión.
- **Automatización Empresarial** — ERP, CRM, facturación, inventarios, dashboards en tiempo real.

---

## Tech Stack

| Tecnología | Uso |
|---|---|
| React 19 + React Router 7 | UI y enrutamiento SPA |
| Vite 8 | Build tool |
| Tailwind CSS 4 + DaisyUI 5 | Estilos |
| GSAP, Motion, AOS | Animaciones |
| EmailJS | Contacto |
| Vercel | Hosting |
